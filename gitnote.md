# Git Setup Instructions

## Quick Setup
Repository URL: https://github.com/g3techvn/g3vnastro.git

## Create a New Repository

```bash
# Initialize with README
echo "# g3vnastro" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/g3techvn/g3vnastro.git
git push -u origin main
```

## Push Existing Repository

```bash
git remote add origin https://github.com/g3techvn/g3vnastro.git
git branch -M main
git push -u origin main
```

## Recommendations
We recommend every repository include:
- README.md
- LICENSE
- .gitignore
